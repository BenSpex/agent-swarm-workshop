# Game Features

## State Scale

- **~120 state variables** in a single flat `GameState` object
- **50+ projects** (one-time upgrades) across all 3 phases
- **BigInt** for values exceeding `Number.MAX_SAFE_INTEGER`: clips, probes, factories, exploredSectors, drifterCount, universalPaperclips
- **localStorage persistence** with auto-save every 25 seconds and manual save/load
- **BigInt-safe JSON serialization** via custom replacer/reviver (JSON.stringify silently drops bigint)

## Phase 1: Business

Key mechanics:
- **Manual clipping** -- click to make one clip at a time
- **Pricing** -- set price per clip; demand is a function of price (demand curve)
- **Wire purchasing** -- buy wire at market price; each clip costs 1 wire
- **Autoclippers** -- produce clips automatically each tick (cost escalates)
- **Megaclippers** -- higher-throughput autoclippers (unlocked via project)
- **Marketing** -- increases demand multiplier (cost escalates)
- **Trust** -- earned at clip milestones (1K, 2K, 5K, etc.), allocated to processors or memory
- **Processors** -- generate operations per second
- **Memory** -- increases max operations capacity
- **Operations** -- currency for purchasing projects
- **Projects** -- one-time upgrades: unlock features, improve formulas, change game mechanics
- **Investment** -- unlock stock/bond portfolio, set risk level, earn returns
- **Creativity** -- generated by quantum computing project, currency for advanced projects
- **Strategic modeling** -- projects that reference real math problems (Hadwiger, Toth Sausage Conjecture)

Key Phase 1 projects (partial list):
- AutoClippers, MegaClippers, Improved AutoClippers, Even Better AutoClippers
- Improved Wire Extrusion, Optimized Wire Buying
- Marketing upgrades (New Slogan, Catchier Slogan, etc.)
- Quantum Computing, Photonic Chip, Strategic Modeling
- Limerick generation, Lexical Processing, Combinatory Harmonics
- Hostile Takeover, Full Monopoly, Xavier Re-initialization
- Space Exploration (triggers Phase 2)

## Phase 2: Earth Consumption

Key mechanics:
- **Harvester drones** -- collect raw matter from Earth
- **Wire drones** -- convert matter into wire
- **Clip factories** (bigint) -- mass-produce clips at industrial scale
- **Solar farms** -- generate power
- **Batteries** -- store power for later use
- **Stored power** -- consumed by drones and factories
- **Momentum** -- drives swarm computing efficiency
- Phase transition: triggered when Earth's resources are fully consumed

Key Phase 2 projects:
- Solar Farm efficiency upgrades
- Harvester/factory throughput upgrades
- Momentum-based computing improvements
- Swarm coordination projects

## Phase 3: Universe

Key mechanics:
- **Probes** (bigint) -- self-replicating space probes
- **Probe trust** -- allocated between 4 stats (speed, exploration, selfReplication, combat)
- **Exploration** -- probes explore sectors of the universe
- **Explored sectors** (bigint) -- count of mapped universe regions
- **Drifters** (bigint) -- encountered entities, some hostile
- **Combat** -- probes fight hostile drifters
- **Honor** -- earned through combat, affects late-game projects
- **Prestige** -- reset the game with accumulated bonuses; universalPaperclips (bigint) tracks total across runs

Key Phase 3 projects:
- Probe launch improvements
- Combat strategy upgrades
- Exploration efficiency
- Prestige/endgame triggers

## UI Features

- **Responsive layout** -- multi-column grid that adapts to phase
- **Message log** -- scrolling feed of game events (new unlocks, milestones, warnings)
- **Phase transition overlays** -- dramatic full-screen transition when entering Phase 2 or 3
- **Notification toasts** -- ephemeral alerts for purchases, achievements
- **Project list** -- filterable by category, shows cost/availability/description
- **Investment panel** -- portfolio value, risk slider, stock/bond allocation
- **Probe control panel** -- trust allocation sliders, launch button, sector map readout
